Infix to Postfix

Small project to covert equations from infix to postfix

This was my favorite project from Computer Science 228, Data Structures. It is a very small program that I wrote in maybe an hour or so. All it does is translate an input.txt that is formatted how a human would normally write mathematical equations and put them into postfix, which is a tree-like method of looking at the equation. The output is written to an output.txt file. I did the extra part of the assignment as well, which was to include parenthesis handling in the code as well. Overall, just a very cool assignment in a class that taught me a lot.
