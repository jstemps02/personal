Backend App Development

REST API and Database code for a Sceduling Application on Android.

This was a group project in Computer Science 309, Application Development, in which my team of 4 decided to make a To-Do list like application that also allowed people to share a list of things to do with one another. I worked on the backend with one other person and really enjoyed setting up APIs. I also attempted to make the application a little more cyber secure while keeping traffic going in plain text by encrypting passwords in traffic and on the backend. Overall, I am happy with how the backend turned out. The frontend of the application was a bit of a mess, but the main functionality is working, and I am very proud of what we have accomplished. For my efforts, I won a best manager award for my teamwork and skill at helping the frontend call the API and getting the connection between the two made.
